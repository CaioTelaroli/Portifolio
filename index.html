Sobre mim

Me chamo Caio Telaroli, estudante de Ciências da Computação pela Estácio e desenvolvedor web Front-end. Tenho foco em desenvolver sites responsivos e agradáveis ao usuário, utilizando meus aprendizados em UX. Trabalho realizando suporte técnico à domicilio, com serviços de manutenção de computadores e consoles, o que me fez aprender a lidar com prazos de forma organizada. Atualmente estudo para entrar no mundo dev, possuindo interesse na área de Front-end com foco em tecnologias JavaScript. Também tenho interesse em me tornar um desenvolvedor Full Stack, utilizando técnologias Java e PHP.

Conhecimentos

HTML5

HTML5 é uma linguagem de indentação utilizada para marcar e contruir os elementos de uma página.

CSS3

CSS3 é uma linguagem de estilização utilizada para estilizar os elementos de uma página.

JavaScript

JavaScript é uma linguagem de programação rápida e multiplataforma, utilizada para criação de aplicativos.

React.js

React é uma biblioteca modular, o que significa que os componentes podem ser facilmente reutilizados e compartilhados entre diferentes partes da aplicação. Seu objetivo é tornar as interfaces eficientes para a experiência do usuário

Java

Java é uma linguagem de programação voltada para a criação de aplicativos.

PHP

PHP é uma linguagem de script de código aberto e de uso geral. Especialmente adequada para o desenvolvimento web e que pode ser embutida dentro do HTML.
